# Engineering Standards

**Project:** PastelTasks
**Status:** Draft
**Last Updated:** 2026-06-26

This frozen specification defines baseline engineering expectations for the project.

## Standards

- Keep implementation modular and aligned with the `lib/app`, `lib/core`, `lib/shared`, and `lib/features` boundaries.
- Prefer offline-first behavior and local persistence by default.
- Keep user-facing flows responsive and accessible.
- Add focused tests for business logic, data access, and critical UI behavior.
- Update living documentation as implementation decisions evolve.
- Treat approved specification documents as baselines unless a new frozen version is intentionally created.
